requests 库的安装

Mac 下使用 Pycharm 时只需按住快捷键 Command + , 选择 Project→Project Interpreter 点击 + 号后搜索 requests 进行安装即可：

获取过程使用的方法

  • requests.get(url, params=None, **kwargs)
    url 是请求的链接，params 是发送请求时参数，kwargs 是可选参数。
